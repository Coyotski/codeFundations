## ⚖️ 1.4 ÁRBOL DE DECISIONES POR CHECKPOINT

### EVALUACIÓN SEMESTRAL - SISTEMA DE PUNTUACIÓN

Cada 6 meses, responde estas preguntas para decidir si continuar, mantener o retroceder:

#### BLOQUE A: SALUD Y BIENESTAR (40% del peso total)
```
□ ¿Presión arterial en rango normal? (120/80 ± 10)
□ ¿Análisis de sangre óptimos? (colesterol, glucosa, hígado, riñones)
□ ¿Dolor articular manejable? (escala 0-3 de 10)
□ ¿Calidad de sueño buena? (7-9 horas reparadoras)
□ ¿Energía diaria alta? (sin fatiga crónica)
```

#### BLOQUE B: RENDIMIENTO Y FUERZA (30% del peso total)
```
□ ¿Progresión constante en los 3 grandes? (+5-10kg cada 6 meses)
□ ¿Técnica mantiene calidad con peso aumentado?
□ ¿Sin lesiones recurrentes?
□ ¿Recuperación entre sesiones completa?
```

#### BLOQUE C: ESTILO DE VIDA (20% del peso total)
```
□ ¿Presupuesto alimentación sostenible?
□ ¿Tiempo para preparar/comer suficiente?
□ ¿Apoyo familiar/social?
□ ¿Impacto laboral manejable?
□ ¿Satisfacción personal con físico actual?
```

#### BLOQUE D: OBJETIVOS PERSONALES (10% del peso total)
```
□ ¿Deseo de competir en powerlifting?
□ ¿Objetivo estético vs funcional claro?
□ ¿Compromiso a largo plazo confirmado?
```

### INTERPRETACIÓN DE RESULTADOS:

| Puntuación | Decisión | Acción |

| 18-20 ✓ | **VERDE** | Continuar subiendo de categoría |
| 14-17 ✓ | **AMARILLO** | Mantener categoría actual 3-6 meses más |
